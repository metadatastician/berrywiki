# Welcome to BerryWiki

BerryWiki is a deterministic, high-performance wiki engine built on a Rust core with a Zig service API (`zig-unified-hexdeca-api`).

## Core Features
- **Semantic Knowledge Graph**: Fully integrated RDFa and JSON-LD for rich metadata querying.
- **SEO Optimized**: Pre-configured with `git-seo` to ensure maximum search engine visibility.
- **Deterministic I/O**: The underlying `berrywiki-core` guarantees atomic, predictable updates.
- **Broken Link Auto-Healer**: Automatically patches references when pages are renamed or moved.
- **Obsidian-style Graph View**: Visualize connections across your entire estate.

## Getting Started
To edit this wiki, simply clone it locally or use the web interface.
